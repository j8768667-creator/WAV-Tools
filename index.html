How to use

  1. Import – select audio files (WAV, MP3, etc.)
  2. Analyse – BPM is estimated automatically
  3. Settings – output format, target BPM, normalise, trim, etc.
  4. Rename – batch rename using tokens and presets
  5. Slicing – chop audio into hits, beats, fixed slices, or find a seamless loop
  6. Loop Builder – arrange slices on a BPM timeline, edit per-slice, generate random patterns
  7. Process – convert, time‑stretch, and encode
  8. Export – download ZIP of all processed files
  9. Converter – convert audio files between WAV and MP3

All processing is local. No files are uploaded. Works offline.
