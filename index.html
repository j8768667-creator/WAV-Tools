How to use

  1. Import – select audio files (WAV, MP3, etc.)
  2. Analyse – BPM is estimated automatically
  3. Settings – output format, target BPM, normalise, trim, etc.
  4. Rename – batch rename using tokens and presets
  5. Slicing – chop audio into hits, beats, fixed slices, or automatically find a seamless loop
  6. Process – convert, time‑stretch, and encode
  7. Export – download ZIP of all processed files

All processing is local. No files are uploaded. Works offline.
